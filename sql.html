Por onde começar com SQL e Banco de Dados?

SQL é a linguagem padrão para a manipulação e extração de dados dos Bancos de Dados relacionais. Esses bancos de dados são estruturados em tabelas, com colunas e linhas, lembrando um pouco as planilhas.

Para começar com banco de dados, você vai estudar como modelar seus dados e saber inserir e localizar informações nas tabelas. Isso vai acontecer através dos comandos SQL como SELECTs e INSERTs. Isso aparece com frequência nesses cenários:

  1. dar o primeiro passo para além do Excel e organizar seus modelos de dados
  2. armazenar dados dos usuários que vem através de sistemas web ou mobile
  3. minerar informações de diversas fontes para fazer análises exploratórias

Os bancos de dados mais populares são: MySQL, PostgreSQL, Oracle e SQLServer.

Além disso, vai encontrar como trabalhar com essas bases usando Python, técnicas de data science e bancos NoSQL.
