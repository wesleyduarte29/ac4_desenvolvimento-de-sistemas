Por onde começar com Java?

Java Básico

Fazer o download do Java, instalar o JDK (kit de desenvolvimento, que vem com o compilador, APIs e ferramentas) e fazer a configuração do ambiente para um simples hello world são passos que podem traumatizar quem está começando. Vamos te ajudar.

Você deve começar por esses primeiros passos e ter uma base sólida em conceitos de maneira prática:

  1. orientação a objetos: focar em praticar sem cair em regras engessadas, como de getter and setter
  2. encapsulamento, escondendo o funcionamento interno das classes e isolando a comunicação em interfaces
  3. reutilização de código, com o uso de bibliotecas importantes


Java Avançado

Aqui entram as tecnologias do Spring Boot e Java EE e também os chamados frameworks, onde escreveremos código de acordo com algumas regras e interfaces para poder ter diversos ganhos em servidores, como o Tomcat, o jetty e o JBoss / Wildfly.

  1. API e Microservices - estamos vivendo na época de APIs e serviços. Aprenda construir uma API Rest com Spring
  2. Boot e use a arquitetura de Microsserviços
  3. Design Patterns: detectar onde faz sentido aplicar as soluções de orientação a objetos com padrões de projetos consolidados
  4. Android - apesar de não ser realmente Java, a linguagem é amplamente utilizada, juntamente com Kotlin
  5. Certificação Java - é uma das provas mais conhecidas entre programadores, com testes de múltipla escolha da Oracle
